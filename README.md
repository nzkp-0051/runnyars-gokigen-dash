# らんにゃーず！ ごきげんダッシュ

猫の「らんにゃー」を12ターン育てて、3ターンごとに発生する「らんにゃーレース」に挑戦するミニ育成ゲームです。

HTML / CSS / JavaScript / localStorage だけで動く、GitHub Pages 向けの静的サイトとして制作しました。

- 公開URL: https://nzkp-0051.github.io/runnyars-gokigen-dash/
- リポジトリ: https://github.com/nzkp-0051/runnyars-gokigen-dash

## 制作目的

個人開発のMVPとして、短時間で最後まで遊べるブラウザゲームを作ることを目的にしました。

育成、レース、卒業、次世代への応援ボーナスという一連の流れを小さく実装し、静的サイトだけで完結する構成にしています。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

フレームワーク、バックエンド、ログイン、DB、外部APIは使っていません。

## 主な機能

- らんにゃー名の入力
- 12ターン制の育成
- 4種類の行動選択
- すばやさ、たいりょく、きあい、ごきげん、つかれの管理
- 3、6、9、12ターン目のらんにゃーレース
- レーススコアによる順位判定
- にぼし獲得
- 卒業画面
- 次世代への応援ボーナス
- 歴代最高にぼし、歴代最多1位回数の記録
- localStorageによる保存、つづきから、データリセット

## 遊び方

1. タイトル画面で「はじめから」を押します。
2. らんにゃー名を入力して「育成開始」を押します。
3. 毎ターン、以下の行動を1つ選びます。
   - ダッシュあそび: すばやさ +5 / つかれ +15
   - おさんぽ: たいりょく +5 / つかれ +15
   - じゃれあい: きあい +5 / つかれ +15
   - おひるね: つかれ -30
4. 3、6、9、12ターン目には、行動後にらんにゃーレースが発生します。
5. 12ターン目のレース後、らんにゃーは卒業します。
6. 卒業時の最終能力と総獲得にぼしから、次のらんにゃーへの応援ボーナスが決まります。

## レースの仕組み

レースでは、他のらんにゃーとの直接比較ではなく、育成したらんにゃーのレーススコアに応じて順位を決めています。

```js
レーススコア =
  すばやさ * レースごとの係数
  + たいりょく * レースごとの係数
  + ごきげん補正
  - つかれペナルティ
  + きあいで下振れしにくくなるランダム値
```

きあいはスコアに直接加算せず、ランダム値の最低値を引き上げる形で使っています。

## 主要なデータ構造

`main.js` の `gameState` がゲーム全体の状態です。

```js
{
  player: {
    totalGraduates: 0,
    totalRaces: 0,
    totalWins: 0,
    totalNiboshi: 0
  },
  runnyarCount: 1,
  supportBonus: { speed: 0, stamina: 0, guts: 0 },
  records: { bestNiboshi: 0, bestWins: 0 },
  currentRunnyar: null,
  lastRaceResult: null,
  lastGraduation: null
}
```

育成中のらんにゃーは `currentRunnyar` に入ります。

```js
{
  name: "ミケまる",
  generation: 1,
  turn: 1,
  speed: 30,
  stamina: 30,
  guts: 30,
  mood: 0,
  fatigue: 0,
  totalNiboshi: 0,
  raceCount: 0,
  winCount: 0,
  raceHistory: []
}
```

## 主要な関数

- `createFreshState()`: 新しい保存データの初期状態を作ります。
- `saveGame()`: `localStorage` に現在の状態を保存します。
- `loadGame()`: `localStorage` から保存データを読み込みます。
- `createRunnyar(name)`: 新しいらんにゃーを作成します。
- `applyAction(actionId)`: 選んだ行動を反映し、必要ならレースへ進みます。
- `rollMoodChange(fatigue)`: 行動後のごきげん変動を計算します。
- `calculateRaceScore(race, runnyar)`: らんにゃーレースのスコアを計算します。
- `runRace(race, runnyar)`: レース結果、順位、獲得にぼしを決定します。
- `calculateSupportBonus(runnyar)`: 卒業後の応援ボーナスを計算します。
- `showTitleScreen()`: タイトル画面を表示します。
- `showCreateRunnyarScreen()`: らんにゃー作成画面を表示します。
- `showTrainingScreen()`: 育成メイン画面を表示します。
- `showRaceResultScreen()`: レース結果画面を表示します。
- `showGraduationScreen()`: 卒業画面を表示します。

## 工夫した点

- 初心者でも追いやすいように、4ファイルだけで構成しました。
- 画面切り替えを `showTitleScreen()` などの関数に分け、処理の入口を見つけやすくしました。
- スコア計算、順位判定、応援ボーナス計算を関数化しました。
- スマホでも押しやすいように、ボタンとカードを大きめにしました。
- localStorageだけで、つづきから遊べる状態を保存しています。

## 動作確認手順

1. `index.html` をブラウザで開きます。
2. 「はじめから」を押して、らんにゃーを作成します。
3. 12ターン分、行動を選びます。
4. 3、6、9、12ターン目にレース結果画面が出ることを確認します。
5. 12ターン目のレース後に卒業画面が出ることを確認します。
6. 「次のらんにゃーを育てる」を押し、応援ボーナスが初期能力に加算されることを確認します。
7. ブラウザを再読み込みして「つづきから」で再開できることを確認します。
8. 「データリセット」で保存データが消えることを確認します。
